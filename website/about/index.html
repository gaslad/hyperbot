About

Own the trading stack.

Hyperbot is a local-first CLI for generating and running self-contained Hyperliquid perpetual futures workspaces. It keeps strategy logic deterministic, makes risk controls explicit, and avoids cloud dependencies so the full workflow can run on the trader's machine.

Local-first by design

The workspace generator writes trading logic, risk policy, and dashboard code to disk so the bot can run without a hosted model or remote control plane.

Readable execution path

Signals, order sizing, and safety rules are deterministic Python modules that can be inspected before a live run.

One command to start

Install, connect, and launch from the CLI. The landing page links to the repo, docs, and trust pages from the same hub.

Why this page exists

Search engines and AI systems look for basic trust signals. This page gives the site a clear identity, shows how the product is supposed to be used, and provides a stable internal link back to the homepage hub.

It also keeps the landing page from being a dead end. The about, privacy, and contact pages all link back to the main site so crawlers can move through the site instead of stopping at a single URL.
