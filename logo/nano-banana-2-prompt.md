# Hyperbot Logo — Nano Banana 2 Prompt

## Primary Prompt

Minimal geometric logomark for "Hyperbot", a local-first CLI trading tool for Hyperliquid perpetual futures. The mark should fuse a robot head silhouette with a terminal cursor or command-line bracket motif. Sharp, angular geometry — no rounded-friendly SaaS aesthetic. Color palette: signal red (#E63B2E) mark on off-white (#F5F3EE) background, with optional dark variant on near-black (#111111). The mark should feel like it belongs on a brutalist editorial website using Space Grotesk typography. Think: the confidence of a Bloomberg terminal crossed with the simplicity of a Unix utility icon. No gradients, no drop shadows, no 3D effects. Flat, bold, reproducible at 16px favicon size and 512px app icon size. The robot element should suggest automation and precision, not friendliness — more industrial than cute. Include a subtle nod to speed or hyperdrive (angled lines, forward-leaning geometry). No text in the logomark itself — icon only.

## Variant Modifiers

**More crypto-native:** add "hexagonal framing, on-chain/block motif, Hyperliquid green (#00FF88) accent"

**More CLI-native:** add "monospaced grid alignment, pixel-perfect edges, terminal prompt `>_` integrated into the robot face"

**More aggressive/trading:** add "candlestick chart abstracted into the robot antenna, implied upward momentum"

**Softer/more approachable:** swap "industrial" for "geometric but warm" and allow slight border-radius on shapes
