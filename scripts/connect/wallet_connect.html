Connect to Hyperliquid

Link your Hyperliquid account so Hyperbot can trade on your behalf via a secure agent wallet.

Uses Hyperliquid's built-in API wallet system. The agent can only trade — it cannot withdraw your funds.

API Wallet Setup

  1. Open app.hyperliquid.xyz/API
  2. Click Create API Wallet (or use an existing one)
  3. Copy your API Private Key and Wallet Address
  4. Paste them below
Your private key is stored in your macOS Keychain. It never leaves your machine or gets sent to any server besides Hyperliquid.
